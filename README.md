# SyntaxWear

SyntaxWear is a modern, responsive e-commerce website template for shoes and sneakers. This project was developed as part of the DevQuest 2.0 course.

## Description

This is a static website built with HTML and CSS, showcasing a fictional online shoe store. The layout is designed to be visually appealing and user-friendly, with a focus on product presentation.

## Features

*   **Responsive Design:** The layout adapts to different screen sizes, from mobile devices to desktops.
*   **Product Showcase:** A grid-based layout to display featured products.
*   **Category Sections:** Sections for different shoe categories like Casual, Sport, Modern, and Futuristic.
*   **Hero Section:** A prominent hero section to highlight new arrivals or promotions.
*   **Newsletter Subscription:** A form for users to subscribe to a newsletter.
*   **Social Media Links:** Links to social media profiles.

## Technologies Used

*   HTML5
*   CSS3

## Project Structure

The project is organized into the following directories:

*   `css/`: Contains the stylesheets.
    *   `base.css`: Basic styles for the body and common elements.
    *   `reset.css`: Resets default browser styles.
    *   `variables.css`: Defines CSS variables for colors, fonts, etc.
    *   `components/`: Contains styles for specific components like the header, footer, and product grid.
*   `images/`: Contains all the images used in the project.
    *   `banners/`: Hero section banners.
    *   `favicons/`: Favicons for the website.
    *   `icons/`: Icons for social media, user actions, etc.
    *   `logo/`: The website logo.
    *   `products/`: Images of the products.
*   `index.html`: The main HTML file.
*   `README.md`: This file.

## How to Use

To view the website, simply open the `index.html` file in your web browser.
